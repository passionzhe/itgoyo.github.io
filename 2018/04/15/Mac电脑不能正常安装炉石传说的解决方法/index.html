Mac电脑不能正常安装炉石传说的解决方法

Apr 15, 2018 | 阅读
今天脑子发热想在Mac上面安装炉石传说但是下载Mac版的战网不能安装、炉石传说客户端不能安装，打开立即报错。后面在论坛终于找到了解决的方法。其实也就是权限的问题
打开mac 终端 输入如下命令：
国际版战网使用这个：
cd ~/Downloads && chmod a+x Battle.net-Setup.app/Contents/MacOS/Battle.net\ Setup
中国版战网使用这个：
cd ~/Downloads && chmod a+x Battle.net-Setup-CN.app/Contents/MacOS/Battle.net\ Setup

如果你发现命令是错的话，请认真对照一下软件的名称，以及文件夹的名称确保无误。祝你好运
