项目名称

github-markdown-toc

安装

$ wget https://raw.githubusercontent.com/ekalinin/github-markdown-toc/master/gh-md-toc$ chmod a+x gh-md-toc

在Mac中如果没有wget，请使用命令brew install wget安装

安装的目录最好是你存.md文件的目录，这样子就可以直接使用命令行./gh-md-toc xxx.md来生成文章相应的目录

用法

  • 命令加上文件所在目录和文件名 ./gh-md-toc ~/projects/Dockerfile.vim/README.md
  • 命令加上md文件网络地址 ./gh-md-toc https://github.com/ekalinin/envirius/blob/master/README.md
  • 命令加上多个文件
➥ ./gh-md-toc \https://github.com/aminb/rust-for-c/blob/master/hello_world/README.md \https://github.com/aminb/rust-for-c/blob/master/control_flow/README.md \https://github.com/aminb/rust-for-c/blob/master/primitive_types_and_operators/README.md \https://github.com/aminb/rust-for-c/blob/master/unique_pointers/README.md
  • toc目录更新 Just put into a file these two lines:
<!--ts--><!--te-->

and run ./gh-md-toc --insert README.test.md
