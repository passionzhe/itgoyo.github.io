AnkiConnect

Apr 2, 2018 | 阅读

Anki Addon：

该插件用于Anki PC客户端扩展功能，配合Chrome插件《Anki 划词制卡助手》使用。
注意：ankiconnet是最初由Github上的Foosoft开发维护，目前版本已经演进到了4.0以上，有些同学在使用Chrome插件Anki划词助手时，可能会在后台选项处遇到版本不对的黄色警告，那是因为插件只认ankiconnect 2.0版本。所以，只要下载本贴内的ankiconnect(是2.0的)并安装就可以了。
注意：好吧，还有的同学是用firefox的，也不能抛弃了。基于种种原因，其实是不想再维护firefox的划词助手组件了，但是老的0.99.6的FF addon还有人用。也是基于种种原因，那个插件用的ankiconnect特喵的是1.0的。按说改上面那个2.0的版本号也可以凑合用，但是audio标签音频下载有点小问题。就把ankiconnect 1.0也放在文末供下载。

安装方法：

将ankiconnect.py文件复制到Anki的Addon目录下。然后重启Anki
Addon目录可以用 如下图方式找到：

文件下载：

ankiconnect 2.0 for Chrome extension 0.99.7
ankiconnect 1.0 for Firefox addon 0.99.6

主义mac用户要在终端里面执行 
defaults write net.ichi2.anki NSAppSleepDisabled -bool true
